Line breaks

We can separate lines with the help of the line break tag.

      
      <body>
       <h1>My favorite things</h1>
        <p>
        Raindrops on roses<br>
        Whiskers on kittens<br>
        Bright copper kettles<br>
        Warm woolen mittens<br>
        </p>
      </body>

Head element

The "head" HTML element, is a container for information about a webpage, like its title. It goes before the body tag.

      
      <head>
      </head>
      <body>
       <h1>My favorite things</h1>
       <p> Raindrops on <em>roses</em>Whiskers on 
        <strong>kittens</strong> </p>
      </body>

Title

To give a webpage a title, we add the "title" and "/title" tag inside the head tags. Note that the "title" is not visible but it provides information to the browser

      
      <head>
       <title>My webpage</title>
      </head>
      <body>
       <h1>My favorite things</h1>
       <p>
       Raindrops on <em>roses</em>
       Whiskers on <strong>kittens</strong>
       </p>
      </body>

Buttons

To create a button users can click on, we need the opening "button" and closing "/button" tags

      
      <body>
       <p>Subscribe to our Newsletter</p>
       <button>Subscribe</button>
      </body>

We can make texts appear on a button by typing them between the tags. Try typing "Login" between the button tags. Make sure you use an uppercase "L".

      
      <body>
       <h1>Facebook</h1>
       <p>Log in to your account to get started</p>
       <button>Login</button>
      </body>

Links

To start creating a link, we add the text between the "a" and the "/a" tags. This won't highlight the text yet. "a" stands for Anchor tag.

      
      <body>
       <a>Learn to Code</a>
      </body>

To link the text to a website, we add "href=""" along with a Uniform Resource Locator (URL). "href" is short for "hypertext reference"

      
      <body>
       <a href="https://getmimo.com">Learn to Code</a>
      </body>

Attributes

"href" is an attribute. All attributes have two things in common: they provide extra information and they go inside the opening tag. Attributes are added after name of the tag, and before ">" closing sign.

      
      <body>
       <a href="https://getmimo.com">Learn to Code</a>
      </body>

Images

To add images to a webpage, we start with the "img" tag.

      
      <body>
       <h3>Drawing of <em>Earthrise</em></h3>
       <img>
      </body>

Just like "br", "img" is an empty tag. That means it has no closing tag.

      
      <body>
       <h3>Drawing of <em>Earthrise</em></h3>
       <img>
      </body>

To display an image, an image tag needs the "src" attribute. "src" stands for source.

      
      <body>
       <h3>Drawing of <em>Earthrise</em></h3>
       <img src="https://mimo.app/i/earth.png">
      </body>

Style tags

There's an easier way to add the same style to one type of element. We sart by adding "style" tags inside the "head" tags.

      
      <head>
       <style>
       </style>
      </head>
      <body>
       <h3>List of Pharaohs</h3>
       <p>Tutankhamun</p>
       <p>Cleopatra VII</p>
       <p>Ramses II</p>
      </body>

Inside the "style" tags, we specify which element to style with a tag slector. Select the "p" tag.

      
      <head>
       <style>
        p
       </style>
      </head>
      <body>
       <h3>List of Pharaohs</h3>
       <p>Tutankhamun</p>
       <p>Cleopatra VII</p>
       <p>Ramses II</p>
      </body>

When we add "p {" followed by "font-family: Papyrus;", and then "}", we create a CSS rule.

      
      <head>
       <style>
        p {
         font-family: Papyrus;
        }
       </style>
      </head>
      <body>
       <h3>List of Pharaohs</h3>
       <p>Tutankhamun</p>
       <p>Cleopatra VII</p>
       <p>Ramses II</p>
      </body>
